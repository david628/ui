属性
renderTo (类型: id/Dom/jquery Dom)

映射的dom.

cm (类型: Object)

对列进行配置


cm: [{
  header: "时间",//表头显示名称
  width: 136,//列宽度，不设置时则为自动宽度
  align: "left",//左，中，右对齐 (left, center, right) 默认为: left
  sort: true,//单击表头是否进行列排序显示
  dataStore: "time",//显示对应数据源字段,
  editType: "text",//设置该列是否为可编辑,类型为(text, commbo)
  valiFn: function(v, dom, $row) {},//编辑时，自动更改值之前触发该方法，些方法设计用来做编辑时验证作用，v为当前列对应的值, dom为该列内侧dom, $row为该行的jQuery dom
  renderFn: function(v, ds) {return ...;}//自定义显示方法， v为当前列对应的值，ds为当前行对应的所有数据，注：如dataStore不进行配置，则不会对自定义过行xss处理，否则进行xss处理.
}, {
  header: "日志等级",
  width: 90,
  align: "center",
  sort: false,
  dataStore: "level",
  renderFn: function(v, ds) {return ...;}
}]
isNumber (类型: Boolean)(默认值:false)

显示行数.

isCheckbox (类型: Boolean)(默认值:false)

设置行选择与全选功能.

width (类型: Number)

设置宽度.

height (类型: Boolean)

设置高度.

exchange (类型: Number)

设置行拖动排序功能.

pageView (类型: Object)

设置分页配置.


pageView: {
  renderTo: "id-grid-pageView",//映射的dom
  param: {//分页传递的参数载体
    offset: 0,
    limit: 10
  },
  callback: function() {}//分页回调函数
}
方法
load (Array json, Number total(可填)): void

加载数据列表(注：配置分页时，只要写入总数，会自动进行分页处理).

isRowSelect (Dom row): Boolean

判断行是否被选择.

selectRows (Dom row, Boolean isSelect): void

选择行.

setSize (Number width, Number height): void

设置宽和高.

getRowData (Dom row): Object

根据行的Dom节点，获取该行的数据.

getCurrentData (): Array

获取所有行的数据集.

getSelected (): Array

获取当前被选择的行Dom节点.

getSelectedData (): Array

获取当前被选择行的数据集.

getData (Dom row): Object

根据行Dom节点，获取相应的数据.

getRows (): Array

获取所有行Dom节点集

hasRows (): Boolean

判断表格有没有数据.

addRow (): void

动态增加数据(为批量增加设计).

deleteRow (Dom row): void

动态删除数据(为批量删除设计).

initRowCallback (Dom row): void

表格每生成一行记录时，触发该方法(为每条记录进行绑定相关事件或处理方法而设计的).

事件
onRowClick (Event e, Dom target, Object opt)

单击行触发事件

onRowDblClick (Event e, Dom target, Object opt)

双击行触发事件

beforeOnHdCheckbox (Event e): void

点击表头全选checkbox触发事件.

handleHdDown (Event e, Dom target, Object opt)

点击表头触发事件

afterOnTreeClick (Event e, Dom target, Object opt)

点击展开行子层触发事件
